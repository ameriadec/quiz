Configuration rapide

Creez votre session en 2 minutes, une seule fois.

1
Creez une base de donnees gratuite (Firebase)

C'est ce qui permet aux telephones des participants d'envoyer leurs reponses a votre ecran en temps reel.

📋 Marche a suivre (5 minutes, une seule fois) :

1. Allez sur console.firebase.google.com (connectez-vous avec votre compte Google)
2. Cliquez Ajouter un projet → donnez-lui un nom (ex: "quiz-cobit") → cliquez Continuer → Creer
3. Dans le menu a gauche, cliquez Build puis Realtime Database
4. Cliquez Creer une base de donnees → choisissez un emplacement → Demarrer en mode test → Activer
5. Vous voyez une URL en haut du type :
https://quiz-cobit-xxxxx-default-rtdb.firebaseio.com
6. Copiez cette URL et collez-la ci-dessous :
